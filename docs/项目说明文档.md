# ASD儿童地板时光家庭干预辅助系统
## 项目说明文档

---

## 一、产品简介

### 1.1 项目背景

中国有超过300万ASD（孤独症谱系障碍）儿童，他们面临社交沟通困难、重复刻板行为等核心障碍。专业康复机构资源稀缺且费用高昂，家庭干预成为核心场景，但家长普遍面临"不知道怎么做""看不到进步""无法精准反馈"等困境。

### 1.2 产品定位

**一个基于AI多Agent协同的ASD儿童家庭干预辅助系统**，通过智能记录、专业分析、个性化推荐，让普通家长也能像专业治疗师一样精准干预，让每个微小的进步都被看见。

### 1.3 核心价值

- **从"说不清楚"到"精准描述"** - 将家长的口语化观察转化为专业记录
- **从"看不到进步"到"可视化成长"** - 捕捉肉眼难以察觉的微观变化
- **从"不知道怎么做"到"个性化指导"** - 基于孩子独特情况提供专业建议
- **从"医家脱节"到"协同干预"** - 搭建家长与医生之间的数据桥梁

### 1.4 服务对象

- **主要用户**：ASD儿童家长（日常使用）
- **协作方**：残联组织、康复医院、专业医生
- **受益者**：2-8岁ASD儿童

## 二、核心功能

### 2.1 微观变化捕捉系统

**痛点**：家长能观察到孩子的行为，但不知道如何描述，看不到规律和趋势

**解决方案**：

#### 三种记录方式

**语音记录（最推荐）**
```
家长说："今天玩积木时，辰辰突然抬头看了我一眼，大概2秒"

AI自动生成：
- 行为：主动眼神接触
- 持续时间：2秒
- 意义：首次主动发起（里程碑）
- 标签：[眼神接触, 主动互动, 突破]
```

**快捷按钮**
- 游戏界面底部常驻：[😊 微笑] [👀 眼神] [🗣️ 出声] [🤝 互动]
- 家长点击 → AI自动记录时间戳和上下文

**引导式记录**
- 系统通过追问帮家长理清思路
- 提供专业选项，降低描述门槛

#### 三种分析能力

**首次出现检测（里程碑识别）**
- AI分析30天记录，自动识别"第一次主动眼神接触"
- 呈现："🌟 这是情感联结的重要突破"

**频率与趋势分析**
- 对比时间序列："主动微笑频率4周内提升140%"
- 可视化呈现：让家长看到"肉眼看不到的进步"

**关联分析**
- AI发现规律："当游戏在上午进行且不超过15分钟时，主动互动增加60%"
- 建议："把重点游戏安排在上午，控制在15分钟内"

### 2.2 智能游戏推荐系统

**痛点**：不知道玩什么游戏，标准化方案无法适配孩子独特情况

**解决方案**：

#### 个性化推荐引擎
- 基于孩子最新趋势、能力短板、兴趣偏好
- RAG检索游戏知识库（50+专业游戏）
- 多目标优化：既巩固优势，又补足短板

#### 推荐理由透明化
```
基于最近观察：
✓ 眼神接触正在改善（+15%/周），应保持动能
✓ 双向沟通仍是短板，需要强化
✓ 孩子对积木有兴趣，匹配度高

推荐目标：将眼神接触从"被动回应"提升到"主动发起"
```

#### 兴趣点识别
- AI从观察中提取："孩子对水流表现出强烈且稳定的兴趣"
- 证据：洗澡时盯着水龙头5秒+、用手接水重复3次
- 游戏建议："水上漂流游戏，培养共同注意力"

### 2.3 实时语音指引系统

**痛点**：游戏过程中不知道该说什么、做什么

**解决方案**：

#### 分步语音引导
- 游戏前：播报目标、步骤、注意事项
- 游戏中：监听家长反馈，提供即时提示
- 游戏后：总结亮点，给予鼓励

#### 推荐话术 vs 避免话术
**推荐**：
- "看看孩子会怎么做，跟随他的节奏"
- "哇，孩子有反应了！继续保持"

**避免**：
- "看着妈妈/爸爸"（命令式）
- "这样做不对"（否定式）

#### 实时亮点捕捉
- AI检测到孩子微笑 → 立即提示："孩子笑了！这是很好的信号"
- 家长点击快捷按钮 → 系统记录并给予正向反馈

### 2.4 视频智能分析系统

**痛点**：家长看不到孩子细微的表情、眼神、肢体变化

**解决方案**：

#### 多维度AI分析
- 表情识别：微笑、皱眉、惊讶等情绪变化
- 眼神追踪：眼神接触时长、主动/被动
- 肢体分析：互动动作、刻板行为
- 声音分析：发声频率、语调变化

#### 关键时刻标注
- 自动标记重要片段："14:32 首次主动递积木"
- 生成视频剪辑，便于家长回顾和分享

### 2.5 成长可视化系统

**痛点**：家长看不到长期进展，容易失去信心

**解决方案**：

#### 多维度可视化
**雷达图**：六大情绪发展里程碑的能力分布
**时间线**：里程碑事件标注，可播放变化动画
**热力图**：微观进步的频率和强度
**趋势图**：各项指标的增长曲线

#### 成长故事呈现
- 自动生成月度/季度成长报告
- 包含：数据对比、里程碑回顾、视频集锦
- 可导出PDF，分享给医生或留作纪念

### 2.6 医家协同系统

**痛点**：家长无法精准反馈，医生无法了解孩子真实情况

**解决方案**：

#### 结构化医学报告
- 一键生成专业报告（PDF格式）
- 包含：干预时间线、能力变化趋势、阶段性效果
- 医生可快速了解孩子进展

#### 个性化反馈通道
- 医生可查看孩子详细数据
- 给出针对性建议
- 系统将建议转化为家长的行动计划

## 三、产品亮点

### 3.1 理论创新：从理论到实践的桥梁

**学术基础**：
- 地板时光（DIR/Floortime）理论有50年临床实践基础
- 六大情绪发展里程碑 → 可追踪的时序指标
- "跟随孩子兴趣" → AI辅助的兴趣识别引擎

**创新突破**：
- 将抽象理论转化为可操作的数字系统
- 建立标准化的微观观察框架
- 为研究者提供大规模、高质量的干预数据

### 3.2 技术创新：动态State + 多Agent协同

#### 动态State设计
传统系统记录固定字段（姓名、年龄、评分），本系统记录**活的时序数据**：

```typescript
// 传统State - 死板的字段
{
  childName: "辰辰",
  eyeContactScore: 35  // 孤立的数字
}

// 动态State - 时序数据
{
  childTimeline: {
    metrics: {
      eyeContact: {
        dataPoints: [
          {timestamp: "01-01", value: 2, source: "assessment"},
          {timestamp: "01-15", value: 5, source: "videoAI"},
          {timestamp: "01-23", value: 8, source: "gameSession"}
        ],
        analysis: {
          trend: "improving",
          rate: 0.22,
          lastMilestone: "首次主动发起互动"
        }
      }
    }
  }
}
```

#### LangGraph多Agent协同
10个专业Agent各司其职，State驱动的动态工作流：

| Agent | 职责 |
|-------|------|
| 初评估Agent | 建立多维度观察框架 |
| 观察捕获Agent | 多源信息融合（视频+语音+文字） |
| 变化检测Agent | 识别进步和趋势 |
| 推荐Agent | 智能游戏推荐 |
| 实时指引Agent | 陪伴式语音引导 |
| 视频分析Agent | AI视频分析 |
| 总结Agent | 单次干预汇总 |
| 反馈表Agent | 智能问卷生成 |
| 记忆更新Agent | Graphiti融合 |
| 再评估Agent | 纵向对比分析 |

#### 人机协作暂停点
- 关键节点等待家长反馈
- AI不会"自说自话"，确保准确性
- 支持中断恢复，保护用户数据

### 3.3 体验创新：移动端优先 + 零门槛操作

**移动端优先设计**：
- 家长随时随地记录观察
- 语音输入，解放双手
- 快捷按钮，一键记录

**零门槛操作**：
- 不需要专业知识，AI自动转化
- 不需要长篇描述，一句话即可
- 不需要手动分析，AI自动识别规律

**即时反馈**：
- 记录后立即看到分析结果
- 发现里程碑立即庆祝提示
- 趋势变化实时可视化

## 四、技术方案特色

### 4.1 技术架构

```
前端层（移动端）
  React + TypeScript + Ant Design Mobile
  实时语音指引、视频录制、可视化成长展示

后端层（AI多Agent协同）
  Node.js + Express + LangGraph
  10个专业Agent协同工作
  State驱动的动态工作流

数据层（时序记忆）
  SQLite（用户数据）
  PostgreSQL + pgvector（游戏知识库）
  Graphiti（长期记忆、时序追踪）
```

### 4.2 核心技术特性

| 特性 | 实现方式 | 价值 |
|------|---------|------|
| **动态State** | 灵活的时序数据结构 | 捕捉微观变化，支持个性化维度扩展 |
| **LangGraph工作流** | 状态驱动的Agent协同 | 信息连贯，支持中断恢复 |
| **Graphiti记忆** | 时序图数据库 | 自动识别里程碑，分析因果关联 |
| **人机协作** | 关键节点暂停等待家长反馈 | AI不会"自说自话"，确保准确性 |
| **多模态融合** | 视频+语音+文字 | 全方位观察孩子，提升分析准确性 |

### 4.3 AI服务集成

**大语言模型**：OpenAI GPT-4
- 语音转文字、描述结构化
- 建议生成、推理解释

**视频理解**：OpenAI GPT-4V
- 表情识别、行为分析
- 情绪判断、关键时刻标注

**向量数据库**：PostgreSQL + pgvector
- 游戏知识库检索
- 个性化推荐

**时序图数据库**：Graphiti
- 长期记忆管理
- 里程碑识别、因果推断

### 4.4 数据安全与隐私

**安全措施**：
- JWT认证，数据加密传输
- 视频文件访问权限控制
- 敏感数据脱敏处理

**隐私保护**：
- 符合《个人信息保护法》
- 家长同意书机制
- 数据匿名化处理
- 定期数据清理

## 五、应用场景

### 5.1 日常干预场景

**场景1：游戏前准备**
1. 家长打开APP，查看今日推荐游戏
2. 系统基于最新观察推荐"积木传递游戏"
3. 家长点击"开始游戏"，系统语音播报目标和步骤

**场景2：游戏进行中**
1. 家长按照语音指引与孩子互动
2. 孩子微笑时，家长点击[😊]按钮记录
3. 系统实时提示："孩子笑了！这是很好的信号"
4. 游戏过程自动录制视频

**场景3：游戏结束后**
1. 家长点击"结束游戏"
2. 系统上传视频进行AI分析
3. 生成本次游戏总结：亮点、进步、建议
4. 家长填写简短反馈表

**场景4：查看进展**
1. 家长打开"成长历程"页面
2. 看到雷达图、时间线、趋势图
3. 发现："主动微笑频率4周提升140%"
4. 获得信心，继续坚持

### 5.2 医家协同场景

**场景1：复诊前准备**
1. 家长一键导出医学报告（PDF）
2. 报告包含：干预时间线、能力变化、视频片段
3. 医生快速了解孩子进展

**场景2：医生指导**
1. 医生查看详细数据和趋势
2. 发现"双向沟通进入平台期"
3. 给出建议："尝试躲猫猫游戏"
4. 系统将建议转化为下周游戏计划

### 5.3 残联服务场景

**场景1：普惠性服务**
- 残联为辖区内ASD家庭提供系统使用补贴
- 降低家庭干预门槛，让更多家庭受益

**场景2：效果追踪**
- 残联查看辖区内整体干预效果
- 汇总数据，评估服务质量
- 为政策制定提供数据支持

## 七、商业模式

### 7.1 目标用户

**C端用户**：ASD儿童家长
- 2-8岁ASD儿童家庭
- 有家庭干预需求
- 愿意使用数字化工具

**B端合作方**：

- 专业版：包含医生指导服务

- 专业治疗师：300-500元/小时
- 线下培训：几千到几万元

**价值**：

- 提升家庭干预质量，减少对机构依赖
- 长期来看，显著降低整体干预成本
