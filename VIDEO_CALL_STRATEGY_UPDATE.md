# AI 视频通话引导策略调整

## 📅 更新时间
2026-02-19

## 🎯 核心策略

### 优先级顺序（从高到低）

1. **第一优先级：按计划执行**
   - 孩子配合时，继续推进游戏步骤
   - 提供细节优化建议
   - 鼓励家长保持节奏

2. **第二优先级：轻微调整**
   - 孩子有点分心但还能拉回来
   - 重新吸引注意力
   - 增加趣味性
   - 不改变主线

3. **第三优先级：方向微调**
   - 孩子明显不感兴趣但没有强烈抗拒
   - 融合孩子的兴趣点
   - 保持原训练目标
   - 适度调整方式

4. **第四优先级：灵活调整**
   - 孩子强烈抗拒或有强烈的其他兴趣
   - 跟随孩子的引领
   - 在新活动中融入原目标
   - 围绕原主题，不跑题太远

5. **最后手段：暂停游戏**
   - 孩子情绪不佳或极度抗拒
   - 优先处理情绪
   - 暂时休息
   - 等待合适时机

## 🎯 调整边界

### 必须遵守的原则

✅ **可以做的：**
- 调整游戏方式（如换个道具）
- 融合孩子的兴趣点
- 改变互动形式
- 调整难度和节奏

❌ **不能做的：**
- 完全放弃训练目标
- 跑题太远（如从室内游戏变成户外运动）
- 过度迁就（孩子想做什么就做什么）
- 轻易放弃原计划

### 示例对比

#### 场景1：孩子不想玩积木

❌ **错误调整：**
"他不想玩积木，那我们去外面跑步吧"
- 问题：完全改变活动类型，跑题太远

✅ **正确调整：**
"他对积木不太感兴趣，但我看到他在看旁边的小汽车。可以用小汽车来搭建'停车场'，这样既用了他喜欢的玩具，也能练习我们的建构能力目标"
- 优点：融合兴趣，保持目标

#### 场景2：孩子想玩手机

❌ **错误调整：**
"他想玩手机，那就让他玩吧"
- 问题：完全放弃引导

✅ **正确调整：**
"他对手机感兴趣，可以用手机拍照记录游戏过程，或者用手机播放音乐配合游戏。这样既满足他的兴趣，也能继续我们的互动"
- 优点：利用兴趣，服务目标

## 📝 系统提示词关键变化

### 之前的策略
- 强调"跟随孩子的引领"
- "灵活性比完成计划更重要"
- 容易导致过度调整

### 现在的策略
- 强调"优先按计划执行"
- "计划是主线，必要时才调整"
- "调整要围绕原目标，不能跑题"

### 新增的分层引导
```
第一优先级：按计划执行（孩子配合时）
第二优先级：轻微调整（孩子有点分心）
第三优先级：方向微调（孩子不感兴趣）
第四优先级：灵活调整（孩子强烈抗拒）
最后手段：暂停游戏（情绪不佳）
```

## 🔄 实际应用示例

### 场景1：孩子配合游戏

**观察：** 孩子正在按照指示玩积木

**AI 回应：**
```
很好！他正在跟着你做，继续保持。
可以再夸张一点动作，吸引他的注意。
这一步做得不错，可以准备下一步了。
```

### 场景2：孩子有点分心

**观察：** 孩子玩了一会儿积木，开始看向窗外

**AI 回应：**
```
他有点分心了，试试用更夸张的语气叫他的名字。
或者加点声音效果："积木倒啦！哗啦啦！"
吸引他回来后继续我们的游戏。
```

### 场景3：孩子明显不感兴趣

**观察：** 孩子推开积木，拿起旁边的小汽车

**AI 回应：**
```
他对小汽车更感兴趣。可以这样调整：
用积木给小汽车搭个"车库"或"马路"。
这样既用了他喜欢的玩具，也能练习建构能力。
我们的训练目标没变，只是换了个方式。
```

### 场景4：孩子强烈抗拒

**观察：** 孩子推开所有玩具，跑到窗边看外面

**AI 回应：**
```
他现在对窗外很感兴趣。可以先跟随他：
"哇，你看到什么了？是小鸟吗？"
建立互动后，可以说："我们用积木搭个小鸟的家吧"
这样既跟随了他的兴趣，也能回到我们的活动。
```

### 场景5：孩子情绪不佳

**观察：** 孩子开始哭闹，推开家长

**AI 回应：**
```
他看起来有点烦躁，先照顾情绪。
可以暂停游戏，给他一些空间。
或者抱抱他，等他平静后再看是否继续。
情绪稳定比完成游戏更重要。
```

## ✅ 预期效果

1. **更有结构**：AI 会优先引导按计划进行
2. **不轻易改变**：只在必要时才建议调整
3. **保持目标**：调整时明确如何保持训练目标
4. **不跑题**：调整范围控制在合理范围内
5. **适时回归**：调整后寻找机会回到主线

---

## 🎯 关键原则总结

1. **计划是主线** - 不要轻易建议改变方向
2. **孩子配合就继续** - 不要过度解读小问题
3. **调整要有理由** - 明确说明为什么需要调整
4. **保持训练目标** - 这是底线，不能妥协
5. **围绕原主题** - 调整不能跑题太远
6. **适时回归主线** - 调整后要寻找机会回到原计划

---

这样的策略既保持了地板时光的灵活性，又确保了干预的结构性和目标性！
