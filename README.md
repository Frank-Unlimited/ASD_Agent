# ASD 地板时光干预辅助系统

## 项目简介

这是一个基于 AI 的孤独症谱系障碍（ASD）儿童家庭干预辅助系统，通过多 Agent 协同工作，帮助家长在家庭场景中更专业、更有效地开展地板时光（Floortime）干预

系统的核心价值在于：让每个微小的进步都被看见。通过 AI 技术捕捉孩子的微观变化，将家长的口语化观察转化为专业记录，提供个性化的游戏推荐和实时干预指导，并建立起医生、家长、孩子之间的数字化协同桥梁。

## 核心功能

- **智能评估建模**：支持导入医院诊断报告或在线填写权威量表，为每个孩子建立个性化数字档案
- **精准游戏推荐**：基于孩子的能力短板和兴趣偏好，推荐适配的地板时光游戏方案
- **实时干预指导**：提供分步语音引导，帮助家长精准把握干预时机和互动技巧
- **AI 视频分析**：实时捕捉孩子的情绪变化和行为特征，生成结构化过程总结
- **动态进展追踪**：构建完整的干预时间线，可视化呈现孩子的能力变化趋势
- **医家协同报告**：一键生成标准化医学报告，打通家长与医生的沟通链路

## 技术架构

- **前端**：React + TypeScript + Ant Design Mobile
- **后端**：基于 LangGraph 框架的多 Agent 协同系统
- **数据存储**：SQLite（用户数据）+ PostgreSQL + pgvector（游戏知识库）
- **长期记忆**：Graphiti 时序数据管理

## 项目愿景

让专业的 ASD 家庭干预触手可及，让每个家长都能成为孩子最好的治疗师，让每个孩子的成长足迹都被记录、被理解、被看见。
